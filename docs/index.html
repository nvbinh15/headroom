Install in three steps

  1. Download Headroom.dmg from the latest release.
  2. Open the disk image and drag Headroom.app into your Applications folder.
  3. Right-click the app and choose Open the first time. macOS will ask for confirmation; click Open once and it's trusted from then on.

Why right-click? Headroom is currently distributed unsigned. macOS Gatekeeper blocks unknown apps on first launch — right-click → Open is the standard workaround. If macOS still refuses, run xattr -dr com.apple.quarantine /Applications/Headroom.app in Terminal.
