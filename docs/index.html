Install

  1. Open Headroom.dmg and drag Headroom.app into Applications.
  2. Open Terminal and run this command to let macOS launch the app:
    xattr -dr com.apple.quarantine /Applications/Headroom.app
  3. Open Headroom from Applications. It appears in your menu bar.

Headroom is unsigned (no paid Apple Developer ID), so Gatekeeper blocks it by default with “Apple could not verify…”. The xattr command above removes the quarantine flag that triggers that warning. On macOS Sequoia and later, this is the only way to launch — the old right-click → Open bypass no longer works.
